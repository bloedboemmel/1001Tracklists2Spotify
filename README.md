# Automatic 1001Tracklist To Spotify Converter
![alt text](./1001/Tracklists2Spotify.png "1001Tracklist To Spotify")

## Usage
Rename the file _keys.py to keys.py and add your credentials in it. You can get your api keys from [Spotify](https://developer.spotify.com/dashboard).

Choose any tracklist from the [1001Tracklists.com](https://www.1001tracklists.com/).

Then simply run 
```bash
pip install -r requirements.txt
python 1001Spoti.py
```
The script will ask you for the 1001Tracklist-Url. Just paste it in the console and press enter.

### That's it!
The script will create a playlist with the name of the 1001Tracklist and the tracks from the 1001Tracklist will be added to the playlist.

## Details
The tracklists-script is a little modified and updated version of the script from [@leandertolksdorf](https://github.com/leandertolksdorf/1001-tracklists-api].
